# AnoGAN-WGAN-pytorch
This is an UNOFFICIAL implement of  Pytorch based AnoGAN combined with WGAN


### Requirements

```
python>=3.6
torch==1.0.0
torchvision==0.2.2
albumentations>=0.4
```
### Note
- Current version has supported  training for mnist datasets.
